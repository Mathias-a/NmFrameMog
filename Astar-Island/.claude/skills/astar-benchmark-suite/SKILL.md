Run the ad-hoc benchmark suite for one candidate and one frozen dataset version.

## Purpose

Use this skill only for the evaluation lane benchmark step. It produces benchmark reports, not promotion decisions.
The first command below is the only operator-facing ad-hoc benchmark command for Astar Island.

## Allowed inputs

- `dataset_version`, one frozen dataset version under `.artifacts/astar-island/datasets/<version>/`
- `candidate`, one candidate id with saved predictions
- Optional `cache_dir`, default `.artifacts/astar-island`

## Exact commands

```bash
uv run --no-project python -m nmframemog.astar_island evaluate-solution benchmark --cache-dir .artifacts/astar-island --dataset-version <version> --candidate <candidate-id>
PYTHONPATH=. uv run --no-project --with pytest pytest tests/astar -q
```

The benchmark command is the sole ad-hoc benchmark entrypoint. The pytest command is the existing offline quality and CI gate agents should run to validate the benchmark surface without live API access.

Wrapped module target used by the runnable wrapper in this checkout:

```bash
python -m nmframemog.astar_island evaluate-solution benchmark --cache-dir .artifacts/astar-island --dataset-version <version> --candidate <candidate-id>
```

## Artifacts produced

- Machine-readable benchmark report artifacts for the candidate and dataset version
- Replay or benchmark evidence generated by the evaluation modules, without modifying dataset contents

## Evidence paths

- `.sisyphus/evidence/`

## Refusal conditions

- Refuse if the candidate predictions were not produced from one frozen dataset version
- Refuse if the caller asks to use live API calls during offline replay or benchmarking
- Refuse if the caller asks to mix multiple dataset versions in one benchmark handoff
- Refuse if the caller asks this skill to bless or promote a candidate
- Refuse if `uv run --no-project python -m nmframemog.astar_island evaluate-solution benchmark` is not available in the current checkout, report the missing wrapper instead of inventing another command
